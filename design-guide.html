Pattern 01

Brand Tone

Pethroom의 브랜드 아이덴티티 키워드와 디자인 언어의 연결 관계입니다.

Premium

Sharp edge, 절제된 컬러, uppercase 타이포그래피로 하이엔드 이미지 구현. 과도한 장식 금지.

Exclusive

VIP 골드 컬러를 멤버십 전용 UI에만 허용. 일반 사용자와 VIP의 시각적 차별화 유지.

Trustworthy

배송·반품 정보를 명확히 노출. 리뷰 수 및 평점 전면 배치. 과장 없는 정보 전달.

Pattern 02

Do / Don't

Pethroom 스타일을 유지하기 위해 반드시 지켜야 할 규칙입니다.

✓ DO

  • ✓ 버튼에 border-radius: 0 사용
  • ✓ 네비게이션 텍스트를 UPPERCASE로 처리
  • ✓ 골드(#C8A96E)는 VIP 전용으로만 사용
  • ✓ 상품 이미지 배경을 #F0EDE8 (Warm Stone) 사용
  • ✓ 3단 가격 구조(정가→할인가→VIP가) 유지
  • ✓ 뱃지는 용도별 고정 색상 준수
  • ✓ 그림자는 hover/popup 상황에만 최소 사용

✗ DON'T

  • ✗ 버튼에 큰 border-radius 적용 (pill 버튼 금지)
  • ✗ 골드 컬러를 일반 UI 요소에 사용
  • ✗ 유채색(파랑, 초록 등)을 주 강조색으로 사용
  • ✗ 진한 drop-shadow 남용
  • ✗ 과장된 애니메이션 또는 장식 요소 추가
  • ✗ VIP 뱃지를 멤버십 무관 상품에 부착
  • ✗ 서체 혼용 (2종 이상 font-family 사용)
